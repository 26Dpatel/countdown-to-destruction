# 🕹️ Countdown to Destruction  
A fast‑paced FPS project built in **Unreal Engine 5.6**, focused on high‑intensity combat, responsive movement, and modular gameplay systems.

---

## 🚀 Overview
**Countdown to Destruction** is an FPS project developed by **Dhruv Patel**, exploring modern shooter mechanics, animation systems, and scalable gameplay architecture.  
The project is structured using industry‑standard Unreal Engine practices, including Git LFS for asset management and a clean, modular folder layout.

---

## 🎮 Features
- ⚡ Fast, responsive FPS movement  
- 🔫 Modular weapon system  
- 🎯 Animation‑driven aiming, firing, and recoil  
- 🧠 Expandable AI framework (WIP)  
- 🗺️ Level‑ready character and weapon assets  
- 🎨 Clean project structure following UE5 best practices

## 📦 Version Control
This project uses **Git LFS** to store large Unreal Engine assets such as:
- `.uasset`
- `.umap`
- `.png`
- `.wav`
- `.mp4`

This ensures fast cloning and clean version history.

## 🧩 Setup Instructions
1. Clone the repo:
git clone https://github.com/26dpatel/countdown-to-destruction.git

## Code
2. Install Git LFS:
git lfs install

3. Pull LFS assets:
git lfs pull

4. Open the project:
- Double‑click `Shadow_Ops.uproject`  
- Let Unreal rebuild project files if needed
